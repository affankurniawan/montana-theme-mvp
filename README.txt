Montana OS 2.0 MVP Theme

What is included:
- OS 2.0 JSON templates
- Modular sections for homepage, about, contact, blog, article, collection, product, testimonials, and company profile
- Theme settings for logo, colors, fonts, email, WhatsApp, and business hours
- Placeholder content designed from the Montana PRD

Before upload:
1. Zip the folder contents, not the parent folder only.
2. In Shopify Admin: Online Store > Themes > Add theme > Upload zip file.
3. Create pages and assign templates:
   - About Us -> page.about-us
   - Contact Us -> page.contact-us
   - Testimonials -> page.testimonials
   - Company Profile -> page.company-profile
4. Create navigation menus:
   - main-menu
   - footer
5. Update Theme settings:
   - logo
   - favicon
   - colors
   - fonts
   - WhatsApp number
   - contact email
   - business hours
6. Create collections and products.
7. Optional: connect Search & Discovery and analytics apps.

Important technical note:
This theme assumes Shopify remains the storefront source of truth for products and variants.
If Big Seller is used, treat it as the sync layer for stock and order operations, not the UI structure source.
